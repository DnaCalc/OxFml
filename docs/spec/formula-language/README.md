# OxFml Formula-Language Spec Set

This directory contains the canonical OxFml formula-language design documents plus archived empirical planning material.

## Bootstrap Reading Order
1. `OXFML_FORMULA_ENGINE_ARCHITECTURE.md`
2. `OXFML_PARSER_AND_BINDER_REALIZATION.md`
3. `OXFML_NORMALIZED_REFERENCE_ADTS.md`
4. `OXFML_OXFUNC_SEMANTIC_BOUNDARY.md`
5. `EXCEL_FORMULA_LANGUAGE_CONCRETE_RULES.md`
6. `EXCEL_FORMULA_LANGUAGE_CONFORMANCE_MATRIX.csv`

## Canonical documents
- `OXFML_FORMULA_ENGINE_ARCHITECTURE.md`
  Core parse/bind/evaluate architecture.
- `OXFML_PARSER_AND_BINDER_REALIZATION.md`
  parser, green/red, bind, and incremental-rebind realization baseline.
- `OXFML_NORMALIZED_REFERENCE_ADTS.md`
  canonical normalized reference atom and reference-expression baseline.
- `OXFML_OXFUNC_SEMANTIC_BOUNDARY.md`
  promoted OxFunc-facing semantic boundary requirements.
- `EXCEL_FORMULA_LANGUAGE_CONCRETE_RULES.md`
  Excel-facing rule corpus and policy wording.
- `EXCEL_FORMULA_LANGUAGE_CONFORMANCE_MATRIX.csv`
  rule register, evidence posture, and promotion criteria.

## Archive
Historical empirical probe plans and seed manifests that are useful for evidence work but not required startup reading live under `archive/`.

Notable archive contents:
- `archive/EXCEL_FORMULA_LANGUAGE_EMPIRICAL_BASELINES.md`
  wave summaries and dated empirical observations that informed the live rule corpus.
- `archive/EXCEL_FORMULA_LANGUAGE_PASS2_PROBE_PLAN.md`
  retained pass-2 execution planning material.
